# 我的linux日常生活-022-Linux 系統時間

```shell
sudo date -s "2022/02/22 19:09:32"
```

若要查詢硬體時鐘（RTC）的時間，可以使用 hwclock：

若要將系統時間寫入硬體時鐘，可以使用 -w 參數：

```shell
sudo hwclock -w
```

## 系統時區校正

### 先設定時區

```bash
sudo timedatectl set-timezone Asia/Taipei
```
